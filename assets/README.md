# Skinlab UK Assets

Add your branding assets here:

- `banner.png` — Hero banner image (recommended size: 1200×400px)
- `logo.png` — Skinlab UK logo (72×72px or SVG favicon)
- `favicon.ico` — Optional browser favicon

Currently the site uses a text-based placeholder logo (SL initials). Replace with your actual Skinlab UK branding once you have the assets ready.

The site will work immediately without these images, but for the full professional look you'll want to add:
- A high-quality beauty-themed hero banner
- Your Skinlab UK logo
- Optional favicon for browser tab

Recommended image formats: PNG with transparency for logo, JPEG or PNG for banner.
